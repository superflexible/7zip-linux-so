# 7-Zip so library for Linux and macOS

This fork adds a library for Linux and macOS, with C and Pascal (FPC) bindings.

The new work is in the subfolder CPP/7zip/Bundles/Format7zLib

Please see the original 7-Zip GitHub page 
https://github.com/ip7z/7zip

and 7-Zip website: [7-zip.org](https://7-zip.org)
